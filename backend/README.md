# SIMPLE MULTIPLE FILE UPLOADER NEST JS 



> save images and videos  in a cloudinary service 

## Docs

- run the server in dev mode and hit the /docs to learn more about the use 

```bash
# docs
$ npm run start:dev

# endpoint
$ GET http:localhost:3000/docs

```


## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
